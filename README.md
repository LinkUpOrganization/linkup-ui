# LinkUp Client

Клієнтська частина крос-платформної соціальної мережі **LinkUp**, розроблена як односторінковий застосунок (SPA) соціальної мережі з підтримкою відображення та аналізу геолокаційних публікацій.

## Основний функціонал

- **Інтерактивна карта:** Відображення публікацій, кластеризація маркерів, теплові карти (Heatmaps).
- **Стрічка новин:** Фільтрація публікацій (Останні, Найкращі, Підписки).
- **Профіль користувача:** Історія подорожей з візуалізацією маршруту на карті.
- **Соціальна взаємодія:** Лайки, коментарі, створення публікацій з фото та геоміткою.
- **Автентифікація:** Вхід через Email/Password або Google OAuth.

## Технологічний стек

- **Framework:** React 18 + TypeScript.
- **Build Tool:** Vite.
- **Routing:** TanStack Router (File-based routing).
- **State Management & API:** TanStack Query (React Query).
- **UI Components:** Material UI (MUI).
- **Maps:** Leaflet, React-Leaflet, OpenStreetMap.

## Налаштування та Запуск

### 1. Попередні вимоги

Переконайтеся, що у вас встановлено:

- [Node.js](https://nodejs.org/) (версія 18 або вище)
- npm

### 2. Встановлення залежностей

Клонуйте репозиторій та встановіть необхідні пакети:

```bash
npm install
```

### 3. Налаштування змінних оточення (.env)

У репозиторії є файл `.env.example`, який містить приклад необхідних змінних.

1.  Створіть файл `.env` у корені проекту.
2.  Скопіюйте вміст із `.env.example` у `.env`.

Приклад вмісту `.env`:

```env
# URL вашого локального або віддаленого API
VITE_API_URL=https://localhost:5001/api
```

### 4. Запуск у режимі розробки

Для запуску локального сервера розробки:

```bash
npm run dev
```

Застосунок буде доступний за адресою: `http://localhost:3000`

## Створення білду (Production)

Для створення оптимізованої версії застосунку для деплою:

```bash
npm run build
```
